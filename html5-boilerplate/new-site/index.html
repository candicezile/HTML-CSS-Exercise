React

React is a JavaScript library for building user interfaces, primarily maintained by Facebook and a community of developers.

React uses a virtual DOM (Document Object Model) to efficiently update the UI, minimizing DOM manipulation for better performance.

React hooks

Hook Summary
useState This hook allows functional components to have local state. It returns a stateful value and a function to update it.
useEffect This hooks adds the ability to perform side effects in functional components.
useContext This is used to consume values from React context without wrapping the component in a context consumer.
